# hongyan

#### 介绍
hongyan 是一个 终端便利的多功能聚合网站

#### 可以做什么
1. 文本传输
例子：
在命令终端执行（cmd/或者bash）
``` curl 
curl -X POST --location "http://localhost:8080/测试KEY/测试VALUE"
```
即成功设置了一个key为测试KEY，value为测试VALUE的对象
接下来在浏览器访问：http://localhost:8080/测试KEY/
或者：`curl -X GET --location "http://localhost:8080/测试KEY"`
即可查询对应的value
